# Исследование рынка — AI Карта потерь для салонов красоты

---

## 1. Конкуренты

### 1. Marquiz (marquiz.ru)
**Что умеет:** конструктор квизов для лидогенерации, популярен в России. Готовые шаблоны, ветвление вопросов, интеграция с CRM и мессенджерами, аналитика по ответам.

| | |
|--|--|
| **Плюсы** | Большая база шаблонов, русскоязычная поддержка, простая настройка, доступная цена |
| **Минусы** | Универсальный инструмент — нет специализации под салоны красоты, нет скоринга и карты потерь, результат не персонализирован |

---

### 2. Interact (tryinteract.com)
**Что умеет:** платформа для personality-квизов и лидогенерации. Работает с 2014 года, 150 000+ клиентов, 1 млрд просмотров квизов. AI-генерация квиза из сайта за 60 секунд.

| | |
|--|--|
| **Плюсы** | Простейший UX, сильная аналитика, огромная база шаблонов, хорошо работает для email-сегментации |
| **Минусы** | Нет специализации под B2B/салоны, нет скоринга по зонам, интерфейс только на английском |

---

### 3. Outgrow (outgrow.co)
**Что умеет:** интерактивный контент — квизы, калькуляторы, чат-боты, формы. Ориентирован на B2B-маркетинг. 1000+ интеграций, ROI-калькуляторы.

| | |
|--|--|
| **Плюсы** | Самый мощный по интеграциям, хорош для сложных воронок, профессиональный вид |
| **Минусы** | Дорого ($95–600/мес), сложный для новичка, нет русскоязычной поддержки |

---

### 4. involve.me
**Что умеет:** AI-квиз с воронкой продаж, лид-скоринг, 55+ нативных интеграций, приём платежей прямо в квизе. Позиционирует себя как «quiz + CRM в одном».

| | |
|--|--|
| **Плюсы** | Гибкая логика, встроенный лид-скоринг, можно принимать оплату, красивые шаблоны |
| **Минусы** | Сложно настраивать сложные сценарии, нет фокуса на бьюти-нише, дорого для малого бизнеса |

---

### 5. TheSalonBusiness.com — Salon Software Quiz
**Что умеет:** квиз для подбора программного обеспечения для салонов (DIKIDI, Fresha, Mindbody и другие). Отвечаешь на вопросы о своём салоне — получаешь рекомендацию по CRM.

| | |
|--|--|
| **Плюсы** | Единственный нишевый квиз именно для салонов, понятный результат, доверие к бренду |
| **Минусы** | Узкая тема (только выбор CRM), нет диагностики потерь, нет плана действий, только на английском |

---

### Вывод по конкурентам

Прямого аналога нет. Все существующие инструменты либо универсальные (Marquiz, Outgrow), либо нишевые, но не про потери (TheSalonBusiness). **AI Карта потерь — единственный инструмент, который диагностирует именно почему клиенты не записываются** и даёт персональный план исправлений для бьюти-бизнеса.

## 3. UX и сценарии

### Сценарий 1 — «Любопытный владелец»
Человек видит ссылку в Telegram-канале про маркетинг для салонов. Заходит с телефона. Читает заголовок — «Узнай, где ты теряешь клиентов» — и думает: *интересно, это про меня*. Нажимает «Начать». Проходит 10 вопросов за 4–5 минут. Получает результат 48/100 — видит красные зоны. Сохраняет скриншот. Возможно, нажимает CTA.

**Что удобно:** короткие вопросы, понятные варианты, прогресс-бар держит.
**Что бесит:** если вариантов ответа нет подходящего, если нельзя вернуться назад.

---

### Сценарий 2 — «Скептик»
Владелец барбершопа заходит по рекламе. Думает: *очередной лид-магнит, сейчас попросят номер телефона*. Проходит квиз настороженно. Результат 62/100 — читает карту потерь — видит, что слабое место именно мобильная версия, и вспоминает что клиенты жаловались. Доверие выросло. Готов оставить контакт.

**Что удобно:** результат без обязательной регистрации.
**Что бесит:** если результат слишком общий и не попадает в его реальную боль.

---

### Сценарий 3 — «Срочный»
Владелица студии ногтевого сервиса ищет решение прямо сейчас — записи мало, деньги кончаются. Находит квиз через поиск. Проходит быстро, за 3 минуты. Результат 29/100 — критический. Читает план — первый шаг: подключить онлайн-запись. Это конкретно. Нажимает CTA с вопросом «Хотите внедрить это за 30 дней?» — оставляет номер.

**Что удобно:** конкретный план с приоритетами и сроками, а не список рекомендаций.
**Что бесит:** если CTA ведёт в никуда или форма не работает.

---

### Общие UX-наблюдения

- Люди бросают квиз если вопросов больше 12–15
- Иконки и эмодзи у вариантов ответа увеличивают досмотр
- Результат «в баллах» воспринимается серьёзнее, чем текстовый вердикт
- Кнопка «Пройти снова» увеличивает возврат — люди хотят проверить после исправлений
- До 80% пользователей заходят с мобильного — вёрстка должна работать идеально

## 4. Технологии

### Вариант A — Чистый HTML/CSS/JS (текущий)

**Плюсы:**
- Мгновенная загрузка, ноль зависимостей
- Работает офлайн, не нужен сервер
- Легко редактировать, весь код в одном файле
- Бесплатный хостинг на GitHub Pages, Netlify, Timeweb

**Минусы:**
- Нельзя сохранять результаты пользователей
- Нельзя отправить письмо с результатами
- Аналитика только внешняя (Метрика)
- При росте функций код станет сложным

**Цена:** 0 ₽ (хостинг ~150–500 ₽/мес)

---

### Вариант B — Готовая платформа (Marquiz / Typeform)

**Плюсы:**
- Быстрый старт, аналитика из коробки
- Интеграции с CRM и мессенджерами
- Не надо кодить

**Минусы:**
- Нельзя сделать уникальный дизайн (ваш стиль)
- Marquiz: от 2000 ₽/мес, Typeform: от ~2000 ₽/мес
- Зависимость от платформы — если поднимут цену, деваться некуда
- Нет скоринга по зонам в стандартных шаблонах

**Цена:** 1500–5000 ₽/мес

---

### Вариант C — HTML + лёгкий бэкенд (следующий шаг)

Добавить к текущему HTML минимальный бэкенд для сбора лидов: формы через **Telegram-бот** (бесплатно) или сервис **GetForm / Formspree** (бесплатный тариф).

**Плюсы:**
- Сохраняете свой дизайн и код
- Получаете лиды без CRM
- Можно добавить за 1 день

**Минусы:**
- Нет автоматической отправки результатов пользователю
- Нужны базовые знания или помощь разработчика

**Цена:** 0–500 ₽/мес

---

### Рекомендация

На старте — **вариант A** (уже реализован), сразу добавить **Telegram-бота** для сбора заявок из CTA (вариант C). Это бесплатно и займёт один день. Платформы (вариант B) — потеря уникального дизайна и зависимость от чужих ограничений.

## 5. Монетизация

### Как зарабатывают похожие проекты

**Модель 1 — Лид-магнит → консультация (самая частая)**
Квиз бесплатный, результат бесплатный. Монетизация — продажа личной консультации или аудита страницы после результатов. Конверсия: 10–20% прошедших квиз оставляют заявку. Средний чек: 3000–15 000 ₽.

**Модель 2 — Подписка для агентств**
Платформа продаёт белый лейбл маркетинговым агентствам: те встраивают квиз в свои услуги для клиентов. Ежемесячная подписка 3000–10 000 ₽/мес.

**Модель 3 — Партнёрства с сервисами записи**
Квиз рекомендует в плане конкретные инструменты (DIKIDI, Yclients) — за регистрацию по реферальной ссылке платится комиссия 500–3000 ₽. Органично вписывается в план действий.

**Модель 4 — Платный расширенный отчёт**
Базовый результат бесплатно, PDF с детальным разбором и персональными рекомендациями — за 490–990 ₽. Низкий барьер, высокая ценность.

**Модель 5 — Курс / гайд**
После результатов предлагается курс «Как самостоятельно исправить карту потерь за 30 дней» — 2000–7000 ₽. Подходит как второй шаг после консультации.
